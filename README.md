# 🌱 わかばチャット

FastAPI と HTML / JavaScript を使って作成した、
シンプルな対話型チャットアプリです。

ユーザーの入力内容に応じて、
共感したり質問を返したりする簡単な会話ができます。

---

## 概要

- ユーザーが名前を入力してチャットを開始します
- FastAPI の API にメッセージを送信します
- メッセージ内容（ポジティブ・ネガティブ）に応じて
  ランダムな返答を返します
- ブラウザ上でチャット形式のUIを表示します

---

## 使用技術

### バックエンド
- Python
- FastAPI
- Pydantic
- CORS Middleware

### フロントエンド
- HTML
- CSS
- JavaScript（fetch API）

---

## 機能

- 名前入力によるチャット開始
- ユーザーごとの簡易メモリ管理
- ポジティブ／ネガティブな言葉の判定
- ランダムな共感・質問メッセージ生成
- 吹き出し形式のチャットUI
